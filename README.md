# 🌍 Wanderlust

Wanderlust is a full-stack web application inspired by **Airbnb**.  
It allows users to explore, list, and book unique places to stay around the world.

## ✨ Features
- 🏡 **Property Listings** – Create, edit, and browse stays with photos and details  
- 🔎 **Search & Filters** – Find stays by location, price range, and availability  
- 👤 **User Authentication** – Secure sign-up, login, and session management  
- 💳 **Bookings & Payments** – Reserve stays (integrated with a payment gateway such as Razorpay/Stripe if enabled)  
- 📱 **Responsive Design** – Works smoothly on desktop and mobile

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript, EJS (templating)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Payment Gateway:** Razorpay (optional)
- **Version Control:** Git & GitHub

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) & npm  
- [MongoDB](https://www.mongodb.com/) instance (local or cloud)

### Installation
```bash
git clone https://github.com/<your-username>/wanderlust.git
cd wanderlust
npm install
```
